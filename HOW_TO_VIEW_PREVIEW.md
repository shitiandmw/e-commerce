# 📸 如何查看图片预览

## 方法一：VS Code（推荐）

1. 在 VS Code 中打开 `COHIBA_IMAGE_UPDATE_PREVIEW.md`
2. 按 `Cmd+Shift+V` (Mac) 或 `Ctrl+Shift+V` (Windows) 打开预览
3. 或者点击右上角的预览图标 📖

## 方法二：Typora

1. 用 Typora 打开 `COHIBA_IMAGE_UPDATE_PREVIEW.md`
2. 图片会自动显示

## 方法三：GitHub

1. 将文件推送到 GitHub
2. 在 GitHub 网页上查看，图片会自动渲染
