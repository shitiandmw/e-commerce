### 1.3 系统架构与数据流

**系统架构**：

```
┌─────────────────┐
│   Admin UI      │  管理后台（Next.js 14）
│  (Port 3002)    │  https://admin.shangjiacigar.com
└────────┬────────┘
         │ REST API
         ↓
┌─────────────────┐
│ Medusa Backend  │  后端服务（Medusa v2.13.1）
│  (Port 9000)    │  PostgreSQL + Redis
└────────┬────────┘
         │ REST API
         ↓
┌─────────────────┐
│ Storefront V2   │  商城前台（Next.js 16）
│                 │  https://www.shangjiacigar.com
└─────────────────┘
```

**核心数据流向**：

| 后台模块 | API 端点 | 前台展示位置 |
|---------|---------|------------|
| 商品管理 | `/store/products` | 商品列表页、商品详情页、搜索结果、首页合集 |
| Banner 管理 | `/store/content/banners` | 首页轮播图 |
| 菜单管理 | `/store/content/menus` | 顶部导航、页脚链接 |
| 商品合集 | `/store/content/curated-collections` | 首页精选区块 |
| 品牌管理 | `/store/content/brands` | 品牌页、商品详情页品牌信息 |
| 文章管理 | `/store/content/articles` | 文章列表页、文章详情页 |
| 公告管理 | `/store/content/home` | 页面顶部公告栏 |

**翻译系统**：
- 所有 API 请求通过 `x-medusa-locale` header 或 `?locale=` 参数传递语言代码
- 支持语言：zh-CN（简体中文）、zh-TW（繁体中文）、en（英语）
- 后端自动返回对应语言的翻译内容

## 第二章：商品管理

### 2.1 模块概述

**功能作用**：
商品管理是电商平台的核心模块，用于管理商城所有商品的信息、价格、库存、变体等。

**影响前台**：
- 商品列表页（分类页、搜索结果页）
- 商品详情页
- 首页商品合集区块
- 购物车和结账流程

### 2.4 数据流向

| 商品数据字段 | API 端点 | 前台展示位置 |
|------------|---------|------------|
| 标题 (title) | `/store/products` | 商品卡片标题、详情页标题 |
| 副标题 (subtitle) | `/store/products` | 商品卡片副标题、详情页副标题 |
| 描述 (description) | `/store/products` | 详情页「商品描述」标签页 |
| 缩略图 (thumbnail) | `/store/products` | 商品卡片封面图 |
| 商品图片 (images) | `/store/products` | 详情页图片画廊 |
| 变体 (variants) | `/store/products` | 详情页变体选择器、价格显示 |
| 价格 (price) | `/store/products` | 商品卡片价格、详情页价格 |
| 库存 (inventory_quantity) | `/store/products` | 详情页库存状态 |
| 品牌关联 | `/store/products?fields=*brand` | 详情页品牌信息、品牌页商品列表 |
| 分类关联 | `/store/products?fields=*categories` | 分类页商品列表、面包屑导航 |
| 标签关联 | `/store/products?fields=*tags` | 商品卡片标签、筛选器 |
| SEO 信息 (metadata.seo) | `/store/products` | 页面 meta 标签 |
| 自定义属性 (metadata.attributes) | `/store/products` | 详情页「自定义属性」标签页 |
